# Registration Form

A simple and responsive Registration Form built using HTML, CSS, and JavaScript. This project demonstrates form creation, input validation, and user-friendly design.

## 🚀 Features

- User registration form
- Client-side form validation
- Required field validation
- Email format validation
- Password validation
- Responsive design
- Clean and easy-to-understand code

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
